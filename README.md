# SmashMateCard

**Version: 1.1.0**

スマメイト(smashmate.net)の1日のレーティング対戦戦績をカード画像にして、X(Twitter)に共有できるChrome拡張機能です。

> **注意**: 本ツールはsmashmate.netの非公式ツールです。

## スクリーンショット

![戦績カードの例](store/screenshot_smashmate_page.png)

## 機能

- 今日の勝敗数・勝率を自動集計
- レート変動（開始 → 終了）をグラフ付きで表示
- 戦績カード画像を自動生成
- 画像のコピー・ダウンロード・X(Twitter)への共有
- 4種類のカードテーマ（Dark / Ocean / Crimson / Light）
- 戦績テキストのクリップボードコピー
- デモモード（対戦データがなくても動作確認可能）

## インストール方法

### 1. ダウンロード
1. [こちらのページ](https://github.com/jun-suzuki1028/SmashMateCard) にアクセス
2. 緑色の「Code」ボタンをクリック
3. 「Download ZIP」をクリック
4. ダウンロードしたZIPファイルを右クリック →「すべて展開」で解凍

### 2. Chromeに追加
1. Chromeのアドレスバーに `chrome://extensions` と入力してEnter
2. 画面右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
4. 先ほど解凍したフォルダ（`SmashMateCard-main`）を選択して「フォルダーの選択」をクリック
5. ツールバーにSmashMateCardのアイコンが表示されたら完了

## 使い方

1. smashmate.net にログイン
2. マイページまたはユーザーページを開く
3. ツールバーの SmashMateCard アイコンをクリック
4. 戦績カードが自動生成される
5. 「画像をコピー」「画像をダウンロード」「Xに投稿」から共有

## 注意事項

- smashmate.net へのログインが必要です
- その日の対戦がない場合（前日比: 0）は表示されません
- データはすべてブラウザ内で処理され、外部サーバーへの送信は行いません

## プライバシー

本拡張機能は、ユーザーのプライバシーを尊重します。

- 収集するデータ: なし
- 外部サーバーへの通信: なし（X への共有ボタン押下時のみ x.com を開きます）
- データの保存: なし（表示のたびにページから都度取得します）
- 必要な権限: `activeTab` のみ（開いているタブのデータ取得に使用）
